# romHEX 14 — 用户手册

欢迎使用 **romHEX 14** 官方用户手册。romHEX 14 是一款由 AI 驱动的 ECU 标定软件,完整支持 A2L、HEX、OLS 和 KP 文件格式。

本手册是 romHEX 14 所有功能、对话框和工作流的完整参考,也可下载 PDF 版本。
